Elementor widget for justified and masonry gallery layouts.

== Description ==
Adds a gallery widget with:
- Justified rows layout
- Masonry/grid visual composition
- Frontend lightbox support
- Editor preview assets

Requires Elementor active.

== Installation ==
1. Upload the plugin folder to /wp-content/plugins/.
2. Activate the plugin.
3. Ensure Elementor is installed and active.
4. Add the widget from Elementor editor.
